Os testes funcionais servem para verificar as funcionalidades de um software, ele verifica se esse software está funcionando de acordo com o que foi especificado para ele. Ou seja, os testes funcionais tem como objetivo garantir que o software esteja realizando as funções e operações esperadas. Um exemplo de teste funcional é verificar se uma função do software retorna o resultado correto de uma operação.
Já os testes não-funcionais servem para verificar as características que não estão diretamente relacionadas com as funcionalidades de um software, esse tipo de teste verifica questões como desempenho, usabilidade e segurança. Ou seja, esse teste tem como objetivo verificar as características do software que irão afetar a experiência do usuário. Um exemplo de teste não-funcional é verificar se um dado fornecido pelo usuário está correto (valor não nulo, formatação correta, etc).
